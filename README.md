
# IA EXTREMA - Bot de Vitórias e Empates

Sistema profissional focado apenas em:
- ✅ Vitórias
- 🤝 Empates

## Tecnologias
- Python
- FastAPI
- Telegram Bot
- Machine Learning
- PostgreSQL
- Docker

## Funcionalidades
- IA de previsão
- Scanner live
- Odds de valor
- Painel web mobile
- Telegram VIP

## Instalação

```bash
pip install -r requirements.txt
python app/main.py
```
